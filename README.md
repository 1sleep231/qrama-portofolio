# 🚀 Quinasa Rama — Data Analyst Portfolio

Selamat datang di repositori *source code* untuk web portofolio pribadi saya. Website ini dirancang untuk menampilkan studi kasus data analitik secara komprehensif, mulai dari identifikasi masalah bisnis, *key insights*, hingga *business impact* dan metrik yang dihasilkan.

🔗 **[Lihat Live Portfolio Disini](https://1sleep231.github.io/qrama-portofolio/)**

---

## 🎯 Mengapa Web Portofolio Ini Berbeda?
Alih-alih hanya menampilkan blok kode atau *dataset* mentah, portofolio ini dibangun dengan pendekatan **Business-Centric**. Setiap proyek yang ditampilkan di sini menceritakan sebuah narasi:
1. **Problem:** Masalah bisnis spesifik apa yang sedang diselesaikan?
2. **Key Insights:** Temuan data apa yang tidak terlihat di permukaan?
3. **Impact/Metrics:** Apa dampak nyata dari solusi tersebut (misal: *Recall model*, potensi konversi, optimasi proses)?

## 🛠️ Tech Stack & Tools

**Web Development (Frontend):**
* HTML5 (Semantic Structure)
* CSS3 (Custom Variables, Flexbox/Grid, Animations, Dark UI)
* Vanilla JavaScript (DOM Manipulation, Lightbox, Iframe Toggle)

**Data Analytics & Engineering (Showcased in Projects):**
* **Languages:** Python, SQL, JavaScript
* **Libraries:** Pandas, NumPy, Scikit-learn, LightGBM, GeoPandas, Plotly
* **BI & Viz:** Looker Studio, Power BI, Matplotlib, Seaborn
* **Automation:** n8n, Flask, Supabase, Telegram Bot API

---

## 📂 Struktur Halaman Website

1. **`index.html` (Overview):** Ringkasan eksekutif profil, statistik pengalaman, dan cuplikan proyek unggulan.
2. **`projects.html` (Case Studies):** Penjabaran mendalam tentang proyek analisis data (*Customer Churn, Mall Segmentation, E-Commerce EDA, Retail Dashboard*). Dilengkapi dengan fitur interaktif *Live Looker Studio Embed*.
3. **`experience.html` (Career & Impact):** Dokumentasi pengalaman kerja nyata dan pencapaian kompetisi (termasuk *timeline* dan *slider* dokumentasi).
4. **`extras.html` (Automated Pipelines):** Menyoroti sistem otomatisasi (*Data Pipelines*) yang mengumpulkan, memproses, dan mendistribusikan *insight* secara mandiri tanpa intervensi manual.
5. **`style.css`:** Sistem desain modular yang mengatur tema warna *cyan/dark-mode*, tipografi, dan responsivitas web.
